# React Native Telephone Input

This library is a forked of [React Telephone Input](https://github.com/mukeshsoni/react-telephone-input).

## Usage

```javascript
render() {
  return (
    <View>
      <TelephoneInput placeholder="Number here, please" />
    </View>
  );

```
